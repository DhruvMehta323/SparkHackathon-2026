# Project Title: Creator DNA
This project ensures any creator's own opportunity in the limelight

## Installation: 
  cd "YOUR_DIRECTORY"
  gh repo clone DhruvMehta323/SparkHackathon-2026

_Dependency Installation_
  
  ```bash
  npm install
  ```
  
  This will install:
  - React
  - React Router DOM
  - Lucide React (for icons)
  - All other necessary dependencies
